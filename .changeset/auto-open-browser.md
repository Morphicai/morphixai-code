自动打开浏览器功能

**新功能**：
- 🚀 运行 `morphixai dev` 后自动打开浏览器
- ⚙️ 添加 `--no-open` 选项禁用自动打开
- 📦 添加 `open` 包依赖

**使用方式**：
```bash
# 启动开发服务器并自动打开浏览器（默认）
morphixai dev

# 启动开发服务器但不打开浏览器
morphixai dev --no-open
```

**用户体验**：
- ✅ 启动更流畅，无需手动复制粘贴 URL
- ✅ 支持所有主流操作系统
- ✅ 打开失败时显示友好提示
